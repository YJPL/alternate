alternate

a portfolio theme for Jekyll IV built with Tachyons CSS

since Y2K20.

alternate is a Y2K20 Jekyll theme initially designed for alternatyves portfolio.

Built for Jekyll.

Check out the documentation if you are new to Jekyll.

Styled with Tachyons CSS

You can change the site styling using Tachyons, look for alt-theme in the src folder. Visit Tachyons CSS for additional documentation
